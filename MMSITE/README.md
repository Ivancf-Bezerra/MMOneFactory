# MMSITE

Este projeto foi gerado com [Angular CLI](https://github.com/angular/angular-cli) versão 19.2.24.

## Servidor de desenvolvimento

Para iniciar o servidor local:

```bash
ng serve
```

Depois, abra o navegador em `http://localhost:4200/`. A aplicação recarrega automaticamente ao alterar os arquivos-fonte.

## Geração de código (scaffolding)

Para gerar um novo componente:

```bash
ng generate component nome-do-componente
```

Para ver todos os schematics disponíveis (`components`, `directives`, `pipes` etc.):

```bash
ng generate --help
```

## Build

Para compilar o projeto:

```bash
ng build
```

Por padrão, o build de produção gera artefatos otimizados de performance.

## Testes unitários

Para executar os testes unitários com [Karma](https://karma-runner.github.io):

```bash
ng test
```

## Testes end-to-end

Para executar testes E2E:

```bash
ng e2e
```

O Angular CLI não inclui framework E2E por padrão; você pode escolher o que melhor se encaixar no projeto.
